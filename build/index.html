LaTeX installeren

LaTeX is alleen de codetaal, iedereen is vrij om zijn eigen programma ervoor te maken, en die zijn er ook. Hieronder een lijst, in volgorde van minst naar meeste moeite, maar ook in volgorde van minst naar meeste features:

  1. Account aanmaken op Overleaf.com. Maar: Dit is een commercieel bedrijf!
  2. Legacy: Installeer MiKTeX (of andere distributie) en TeXstudio. Instructies beneden.
  3. Installeer MiKTeX (of andere distributie) en Visual Studio Code. Instructies op vkuhlmann.com/latex/installation.

LaTeX installeren (legacy uitleg)

  • LaTeX is de codetaal die wij je aanleren om mooie bestanden met formules te maken.
  • Overleaf is een website waarop je LaTeX kan schrijven en het als PDF kan zien.
  • TeXstudio is een programma waarin je LaTeX kan schrijven en het als PDF kan zien.
  • MiKTeX is een hulpprograma die TeXstudio nodig heeft.

Overleaf is een fijne site, maar onbetrouwbaar vanwege zijn commerciële karakter. Zo zien we dat er steeds meer functionaliteiten betaald worden. Daarom vragen we je om TeXstudio (+ MiKTeX) te installeren. Als je samenwerkt in groepjes van twee, en tegelijkertijd aan hetzelfde LaTeX bestand wil schrijven, dan moet je hiervoor wel uitwijken naar Overleaf.

Update: VS Code i.p.v. TeXstudio

Deelnemers ergeren zich meer en meer aan het oude uitzien van TeXstudio, en om eerlijk te zijn; ook van een technisch perspectief is het ver achtergebleven ten opzichte van moderne code editors zoals Visual Studio Code. In die laatste kan je ook LaTeX typen, en het programma ziet er een stuk mooier uit.

Omdat het heel lastig is om iets goeduitziends te maken op deze deelwebsite, laat staan om het goed uit te bouwen, onderhoudbaar te houden of interactiviteit toe te voegen, maak in mijn documentatie nu op mijn eigen website.

Je vindt de installatieinstructies op vkuhlmann.com/latex/installation

De installatie-instructies staan onderaan deze pagina. Meer details over de verschillen tussen Overleaf en TeXstudio (+ MiKTeX):

Voordelen van TeXstudio (+ MiKTeX)
  • Altijd en overal: geen goede internetconnectie nodig.
  • In controle over je eigen bestanden: heb alles veilig op je eigen computer, en niemand die kan meekijken.
  • Niet-commercieel, dus nooit limitaties op wat het kan. Gemaakt door de LaTeX-gemeenschap (Projectpagina van TeXstudio).
  • Geen account nodig.
  • Meer keyboard shortcuts en andere geavanceerde features.
Aan de slag! Volg de installatie-instructies beneden.
Voordelen van Overleaf
  • Geen installatie nodig.
  • Werk tegelijk aan een document voor twee personen.
  • Direct al iets dankzij templates.
  • Je bestanden staan automatisch in de cloud, dus zijn overal beschikbaar. Zo heb je dus ook een back-up, handig als je ooit computerproblemen hebt.*
Aan de slag! Ga naar Overleaf.com, maak een account aan en klik op 'New Project'.

* Bij TeXstudio heb je hetzelfde gemak als je je bestanden opslaat in een map die synchroniseert met een online opslagdienst. Lees hoe je dit doet met OneDrive.

2.  TeXstudio (+ MiKTeX) installeren

We gebruiken twee programma's:
  1. Een distributie: Dit voorziet de mechaniek die je code omzet in een PDF. Als je met een command line kan omgaan, dan kan je hiermee al LaTeX-code omzetten in een PDF, maar vaak is het fijner om te werken met een speciale editor.
  2. Een editor: De editor toont je code en verbetert je ervaring door bijvoorbeeld code te kleuren en automatische aanvullingen te geven. Telkens als je op de Compileer-knop duwt geeft het het nodige commando aan de distributie, en toont het de PDF handig naast je code.

Doordat deze twee taken losgekoppeld zijn heb je meer vrijheid om te kiezen welke editor je gebruikt.

De distributie

Installeer eerst de distributie (zodat de editor bij installatie al meteen ziet waar de distributie is):

  • Windows: MiKTeX (alternatief: TeX Live)
  • Mac: MiKTeX (alternatief: MacTeX)
  • Linux: TeX Live (alternatief: MiKTeX)

    Op een Linuxdistributie gebaseerd op Debian (zoals Ubuntu en Linux Mint):

    sudo apt-get install texlive-full texstudio
De distributies verschillen functioneel niet echt van elkaar. MiKTeX kan op Windows wel automatisch packages installeren waar TeX Live dat niet kan (met standaardinstellingen downloadt TeX Live alle packages, dit kost enkele gigabytes opslag). TeX Live gaat wel wat beter via de command line, wat een belangrijke pre is op Linux.

De editor

Op alle drie platformen kan je terecht bij TeXstudio.

Afronding

  1. Eenmaal alles succesvol is geïnstalleerd, zou 'TeXstudio' tussen je programma's moeten staan. Open TeXstudio.

  2. Copy+paste of schrijf het volgende document in TeXstudio:

    \documentclass[a4paper]{article}

    \title{Test}
    \begin{document}
        \maketitle
    \end{document}

  3. Druk op F5 (afhankelijk van je toetsenbord moet je mogelijk ook Fn induwen), of duw op de 'doorspoel'-knop in de toolbar van TeXstudio (links van de 'afspeel'-knop).

  4. Als alles goed is, zou eventjes later rechts van de code een simpel document (enkel titel en datum) moeten verschijnen.

    Alles is niet goed! Dan verschijnt er waarschijnlijk een foutbericht onder je code. Probeer elk van de volgende mogelijke oplossingen:
    1. Controleer dat je code overeenkomt met bovenstaand voorbeeld.
    2. Als je MiKTeX gebruikt, controleer dan dat 'MiKTeX Console' bestaat op je computer.
    3. Herstart je computer en probeer opnieuw.
    4. Probeer Robins exorcisme.
    5. Nog steeds niet? Vraag ons om hulp:  Contact
  5. Het is verstandig om te zorgen dat MiKTeX zijn packages up-to-date heeft. Daartoe open je 'MiKTeX Console' en klik je op Updates > Check for updates. Als er updates worden getoond, klik dan op de 'Update now'-knop.
  6. Gefeliciteerd! Welkom in de wereld van LaTeX!
