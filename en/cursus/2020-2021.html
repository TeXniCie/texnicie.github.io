At 28 September 2020, the yearly LaTeX course will start. In four sessions, every monday at 17:15, the TeXniCie will get you familiar with LaTeX and will teach you the important know-how you'll need in your study – aside from mathematics, you will learn amongst other things about figures and references in LaTeX.

You don't need to know any LaTeX yet. We do however ask you to already install the application for LaTeX yourself, the instructions can be found on our Home page (a-es2.nl/hektex). The LaTeX course will be given completely online due to COVID-19. For this we work on a Team on Microsoft Teams. You don't need to register, just join the Team using the code lz7jzfk. Still questions? Contact us.

Till soon!

Question & Answer

Question: References don't appear in the bibliography. Biber shows this reference as (book, label). How do I fix it?
Answer: Set Biber as the default bibliography manager (see thesis manual). With some older LaTeX distributions Biber is missing. Biber is often installed in C:\Program Files\MiKTeX 2.9\miktex\bin\x64. Is it missing? Update MiKTeX, or, in the worst case, reinstall MiKTeX.
