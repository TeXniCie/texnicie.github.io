3.  Question & Answer

Question: References don't appear in the bibliography. Biber shows this reference as (book, label). How do I fix it?
Answer: Set Biber as the default bibliography manager (see thesis manual). With some older LaTeX distributions Biber is missing. Biber is often installed in C:\Program Files\MiKTeX 2.9\miktex\bin\x64. Is it missing? Update MiKTeX, or, in the worst case, reinstall MiKTeX.
