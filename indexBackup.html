LaTeX-cursus (info voor 2019-2020)

Vanaf 23 september 2019 zal de TeXniCie van A–Eskwadraat een serie van 4 LaTeX-cursussen geven op maandagen van 17 tot 19 uur. Hier kan je de documenten en informatie over deze cursus vinden.

Hoe werkt het?

Het is handig je van tevoren voor de cursus in te schrijven, dit kun je hier doen. Voor de eerste week hoef je geen voorkennis te hebben. We leggen ook uit hoe je LaTeX moet installeren op je laptop. Voor de daarop volgende weken verwachten we dat de studenten de kennis van de weken daarvoor bezitten. Als je dus week 4 wilt volgen verwachten we dat je de technieken van week 1, 2 en 3 kent.

Daar er beperkt computers beschikbaar zijn vragen we je om een eigen laptop mee te nemen. Zeker omdat je dan ook thuis met LaTeX kan werken.

Scriptiecursus (info voor 2019-2020)

De TeXniCie heeft voor jullie tijd besteed om deze mooie scriptie template te maken. Hierin kan je niet alleen zien hoe bepaalde dingen werken die je misschien nodig gaat hebben bij je scriptie, maar je kan het ook als een functioneel begindocument gebruiken. Deze is te vinden op Scriptietemplate handleiding (Versie 2020).

Andere versies:
  • Scriptietemplate handleiding zonder subfiles (Versie 2020); Chaotischer, maar gemakkelijker.
  • Scriptietemplate handleiding zonder subfiles (Versie 2018); Voor terugwaartse compatibiliteit.

Vraag & Antwoord

Vraag: Refenties verschijnen niet de referentielijst, Biber toont de referenties als (book, label). Hoe fix ik dit?
Maak Biber de standaard bibliografiemanager (zie scriptiecursushandleiding). Bij sommige oude LaTeX distributies ontbreekt Biber. Biber staat vaak in: C:\Program Files\MiKTeX 2.9\miktex\bin\x64. Heb je geen Biber? Update MiKTeX, of, in het ergste geval, herinstalleer MiKTeX.

Zelf al installeren

We gebruiken twee programma's:
  1. Een distributie: Dit voorziet de mechaniek die je code omzet in een PDF. Hiermee kan je al via een command line tekstbestanden omzetten in PDF, maar vaak is het fijner om te werken met een speciale editor.
  2. Een editor: De editor toont je code en verbetert je ervaring door bijvoorbeeld code te kleuren en automatische aanvullingen te geven. Telkens als je op de Compileer-knop duwt geeft het het nodige commando aan de distributie, en toont het de PDF handig naast je code.

De distributie

Doordat deze twee taken losgekoppeld zijn heb je meer vrijheid om te kiezen welke editor je gebruikt. Installeer eerst de distributie (zodat de editor bij installatie al meteen ziet waar de distributie is):
  • Windows: MiKTeX of TeX Live
  • Mac: MiKTeX of MacTeX
  • Linux: TeX Live of MiKTeX
De distributies verschillen functioneel niet echt van elkaar. MikTeX kan op Windows wel automatisch packages installeren waar TeX Live dat niet kan (met standaardinstellingen download TeX Live alle packages, dit kost enkele gigabytes opslag). TeX Live gaat wel wat beter via de command line, dus kan het wel de voorkeur hebben van Linux-gebruikers.
